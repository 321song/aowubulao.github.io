Ubuntu18.04安装Docker和酷Q

前言

最近突然想玩一下QQ机器人，发现WEB QQ已经停止支持了，酷q只能在windows系统上运行。不巧的是我前一阵子才把仅有的一台Windows Server的服务器换成了Ubuntu，只能通过Docker中安装wine+酷q的形式运行了。

安装Docker

安装前

sudo apt update

安装

sudo apt install docker.io -y

安装完成后

systemctl start docker
systemctl enable docker

检查Docker版本

docker --version

将Docker添加到非root用户

1、创建新用户添加

useradd -m -s /bin/bash dockeruser
passwd dockeruser

usermod -aG docker dockeruser

hello world!

su dockeruser
docker run hello-world

提示下载后即可得到hello world

2、已有用户添加

也可以参照上面的方法直接用已有的用户添加

安装酷 Q

安装

docker pull coolq/wine-coolq

创建数据文件夹

可以创建在任意路径

mkdir /home/dockeruse/coolq-data

启动

docker run --name=coolq -d -p 7091:9000 -v /home/dockeruse/coolq-data:/home/user/coolq -e VNC_PASSWD=12345678 -e COOLQ_ACCOUNT=123456 coolq/wine-coolq

端口7091可以修改为自己想启动的端口，VNC_PASSWD为VNC链接的密码，COOLQ_ACCOUNT为QQ号，建议在启动容器时定好，以后连接会方便很多。

连接

浏览器访问：服务器IP:端口 即可

一些小坑

可能也不算坑，是我的理解不算太好，如果仔细看的话： 

/home/user/coolq

这个目录是不存在的，虽然酷Q的数据会在我们创建的数据文件夹里，但实际使用的并不是这个文件夹里的数据，上传插件后，结果还是找不到插件，从而开始怀疑人生。

find / -name CQA.exe

我通过这个命令查找后

才发现了真正的数据文件，这样就能正确上传插件了。
